Y-Shop E-Commerce App

YSHOP is a full-stack intelligent commerce platform unifying E-Commerce, CRM, ERP, and advanced AI. The recommendation engine is built on vector embeddings — every product interaction is encoded into high-dimensional semantic vectors stored in a vector database, enabling millisecond-precision similarity search. The system builds a persistent taste profile per user, and the LLM queries the vector store directly rather than relying on keyword matching — producing recommendations that feel eerily accurate. Collaborative filtering emerges naturally from vector proximity: users with overlapping embedding clusters surface each other's preferences automatically. Also integrates real-time AI moderation, a 24/7 voice assistant, and drone delivery via NVIDIA Jetson.

Tech: Vector DB • Text Embeddings • LLM • Node.js • Python • Swift & SwiftUI • Flutter • Firebase • NVIDIA Jetson • PX4 • MAVSDK • OpenCV • Drone Delivery • AI Moderation • Voice Assistant
